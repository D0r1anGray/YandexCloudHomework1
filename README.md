# Проект: Витрина данных mart_city_top_products на PySpark

Данный проект выполнен в рамках домашнего задания по дисциплине "Семинар наставника" (тема: Яндекс Cloud). Целью работы является построение аналитической витрины данных, отображающей наиболее прибыльные товары в разрезе городов с использованием технологий распределенной обработки данных.

## Стек технологий
* **Облачная платформа:** Yandex Cloud
* **Среда разработки:** Apache Zeppelin (интерпретатор `%spark.pyspark`)
* **Движок обработки:** Apache Spark / PySpark (Spark DataFrame API)
* **Хранение данных:** HDFS / S3-совместимое хранилище (формат Parquet)

## Описание решения
В ходе выполнения задания были реализованы следующие этапы обработки данных:

1.  **Генерация данных:** Создание исходных DataFrame для пользователей (`users`), заказов (`orders`) и продуктов (`products`) непосредственно в среде Zeppelin.
2.  **Расчет метрик:**
    * Вычисление выручки для каждой транзакции: `revenue = qty * price`.
    * Обогащение данных путем объединения (join) таблиц по ключам `user_id` и `product_id`.
3.  **Агрегация:** Группировка данных по городу и продукту с расчетом общего количества заказов (`orders_cnt`), суммарного количества проданного товара (`qty_sum`) и общей выручки (`revenue_sum`).
4.  **Аналитика (Window Functions):** Использование оконных функций для ранжирования товаров по выручке внутри каждого города.
5.  **Фильтрация:** Выбор топ-2 наиболее прибыльных товаров для каждого региона.
6.  **Экспорт:** Сохранение финальной витрины в файловую систему HDFS по пути `/tmp/sandbox_zeppelin/mart_city_top_products/` в формате Parquet с применением режима `overwrite`.
